# EPub-js Terminal Reader

Read EPub books in your terminal.

## Usage

| Operation     | Default Key  |
| ------------- | ------------ |
| Next Page     | `j`          |
| Prev Page     | `k`          |
| Next Chapter  | `l`          |
| Prev Chapter  | `h`          |
| Boss Key      | `Space`      |
| Save and Quit | `Ctrl` + `C` |
